桐邻“桐”来自桐梓林，
“邻”代表邻里之间的连接。让附近的人与事，彼此看见。
看

共

从桐梓林出发，让每一位邻居更容易找到彼此与社区

08 / BILINGUAL EXPERIENCE微信与双语

让语言成为入口，而不是门槛

微信小程序 · CloudBase dev 实时数据

正式内容：中英双语维护

活动、地点、公告等使用中文和英文字段，发布前由内容负责人补齐。

居民内容：保留作者原文

帖子和评论不伪造另一语言版本，只标注原始语言并本地化界面。

界面与正式内容：随语言切换

导航、按钮、状态提示和地点正式内容统一切换，UGC 仍保留作者原文。

13 / CURRENT STATE现状与边界

已经具备

  • Events、Discover、Places 三个核心模块
  • 移动端、后台与统一 API 的基本闭环
  • 中英双语界面和正式内容发布门禁
  • CloudBase 开发环境与真实地点/活动验证基础

适合继续进行内容准备、工作人员培训和全流程验收。

公开发布前仍需

  • 微信账号、备案、域名及隐私合规证据
  • 生产配置、安全规则与后台操作员验证
  • iOS / Android 真机验收和内容媒体审计
  • 上传审核、发布审批与上线值守安排

这些事项需要账号负责人、CloudBase 操作员、QA 与发布负责人共同完成。
