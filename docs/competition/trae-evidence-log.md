- **Objective**：该阶段目标
- **Main Files**：主要产出/涉及文件
- **Commit**：关联 commit hash
- **Screenshot**：截图路径（相对 `evidence/` 目录）
- **Result**：该阶段结果（Done / Selected / PASS / BLOCKED / IN-PROGRESS）
