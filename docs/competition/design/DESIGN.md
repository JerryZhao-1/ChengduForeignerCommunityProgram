# 设计总纲 — 桐邻 First 120 Minutes

## 设计目标

将现有桐梓林社区地图 H5 的零散功能（活动、地点、发现），组织成一条面向新到外籍居民的"前 120 分钟融入路线"叙事，降低首次使用认知成本。

## 设计原则

1. **引导优先**：首屏提供步骤化路线，而非功能罗列
2. **双语平等**：中英双语同步呈现，非翻译附加
3. **真实数据**：基于 packages/shared 真实契约，不硬编码 demo 假数据
4. **最小新增**：复用现有页面组，以编排和引导层为主

## 复用现有能力

| 现有能力 | 在 demo 中的角色 |
|----------|------------------|
| home 首页 | 融入路线引导入口承载页 |
| events | 活动发现动线 |
| places（map + detail） | 地点导航动线 |
| discover | 社区连接动线 |
| more/language-settings | 双语切换 |
| i18n catalog | 全程中英双语支撑 |

## 新增设计元素（计划）

- 融入路线引导组件（步骤卡 / 进度指示）
- 首屏叙事文案（中英双语）
- demo 专用入口路由（可选）

## 非目标

- 不做多社区平台化
- 不引入新 UI 组件库（H5 端沿用现有；微信小程序端遵守 TDesign MiniProgram）
- 不改变现有 API contract
