# Computer Use MCP-only GUI runbook

Use Computer Use against the URL printed by `run.sh` / `run.bat`. Do not create or execute Playwright, Selenium, or browser automation scripts.

## Profile matrix

Run each profile in both zh and en and capture plan, route-map, and complete states:

1. `community-service` / `first-week` / `solo` / `none`
2. `food-drink` / `first-month` / `family-with-kids` / `wheelchair`
3. `social` / `settled` / `couple` / `low-vision`
4. `outdoor-sports` / `first-week` / `shared` / `quiet-environment`

## MCP execution assertions

For each fresh state, read the current accessibility tree before selecting element indices. Use Computer Use screenshots as the original GUI evidence.

1. At mobile width, complete welcome → preferences → plan → route-map → complete in under 180 seconds.
2. Inspect every rendered page for clipping or horizontal overflow. Record the tested viewport/window geometry and visual observation.
3. Inspect the accessibility text for raw enum keys (`primary_interest`, `arrival_context`, `household_type`, `accessibility_need`) and raw values (`community-service`, `first-week`, `family-with-kids`, `wheelchair`, `low-vision`, `quiet-environment`). None may appear as user-facing copy.
4. Inspect the accessibility text for AI/model wording (`generated by AI`, `AI-generated`, `AI 生成`, `model status`, `模型状态`, `powered by LLM`, `GPT`, `DeepSeek`, `generated_by`, `ai_status`). None may appear.
5. Switch zh → en → zh on the same generated plan and verify the route content persists.
6. Verify route-map shows the place stop before the event stop and remains usable with the map-unavailable notice.
7. Verify Finish Route stays disabled until Mark visited and Demo Confirm have both been activated; then verify complete shows place 1/1 and event 1/1.
8. Repeat one plan at desktop width and verify centered max-width layout, visible focus, and usable targets.
9. Record Computer Use accessibility observations in `logs/console-observations.md`. If console inspection is unavailable through the configured MCP surface, mark that assertion honestly as not observed rather than inferring it from screenshots.
