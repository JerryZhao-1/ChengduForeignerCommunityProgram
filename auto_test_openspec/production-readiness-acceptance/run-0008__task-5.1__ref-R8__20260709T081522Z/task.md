# Task 5.1 - Discover Content, Interaction, Report, And Governance Acceptance

Change: `production-readiness-acceptance`

Reference: `#R8`

Scope: Mixed

The GUI portion is documented in `tests/gui_runbook_discover_acceptance.md` and must be executed in WeChat DevTools or on a true device.

## Run

```bash
./run.sh
```

Outputs are written to `outputs/`; logs are written to `logs/`.
