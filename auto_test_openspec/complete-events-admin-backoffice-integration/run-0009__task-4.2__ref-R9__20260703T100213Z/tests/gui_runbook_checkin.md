# GUI MCP Runbook: Check-In

Use MCP browser control only.

1. Open `http://127.0.0.1:5173/events`.
2. Open `报名` for `Weekend Neighborhood Brunch`.
3. Fill `ticket_001` in the check-in input and click `核销`.
4. Confirm success message and registration table status changes to `used` with a used time.
5. Capture `outputs/screenshots/task-4.2-success.png`.
6. Submit `ticket_001` again.
7. Confirm an error message appears and no unrelated rows change.
8. Capture `outputs/screenshots/task-4.2-repeat-error.png` and write paths to `logs/screenshots.md`.
