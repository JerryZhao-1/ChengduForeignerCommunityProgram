# GUI MCP Runbook: Events Table And Filters

Use MCP browser control only.

1. Open `http://127.0.0.1:5173/events`.
2. Confirm the table contains both `Weekend Neighborhood Brunch` and `Draft Event`.
3. Capture `outputs/screenshots/task-3.1-table.png`.
4. Set publish status filter to draft and confirm only draft rows remain.
5. Set publish status filter to published and confirm published rows remain.
6. Search keyword `offline` and confirm `Offline Event` appears.
7. Confirm no blocking console errors.
8. Record screenshot paths and console summary in `logs/screenshots.md`.
